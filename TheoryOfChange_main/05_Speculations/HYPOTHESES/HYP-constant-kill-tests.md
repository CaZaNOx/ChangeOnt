# Hypothesis — Kill tests for CO-process constants

Claim (speculative)
- A valid “constant from process” must survive explicit kill tests across media: heavy-tail jump processes (α<2), anisotropic diffusion, curved manifolds (sphere/hyperbolic), and graph Laplacians with lattice artifacts. Failure in any shows the pipeline is smuggling assumptions.

Test sketch
- Implement the constant-estimation pipeline (as in HYP-CO-constants-from-process) and run on the above media. Record whether the scalar converges, drifts, or splits; quantify sensitivity to preprocessing and sample size.

Why it matters
- Forces rigor on “constants from change” by requiring invariance or controlled drift across well-defined counterexamples; otherwise the claim is vacuous.
