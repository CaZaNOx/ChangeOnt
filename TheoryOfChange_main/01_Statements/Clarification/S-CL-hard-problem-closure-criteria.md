# Closure requires intersubject transport, necessity proofs, and gauge bounds

AI_13 concludes (lines 17130‑17166) that the hard problem collapses only if three criteria are met: (1) demonstrate commuting, low-error intersubject transport in the canonical latent space (low STE under interventions, showing sufficiency up to warp); (2) prove necessity on toy systems and approximate sufficiency via experiments that manipulate the thick-present tuple; and (3) bound the residual gauge freedom (“this-ness”) so what remains is an indexical coordinate choice, not an extra ontic fact. If a theorem establishes Phenomenality \(\iff \mathcal{J}\) (realizing the tuple with constrained gauge) and these experimental requirements hold, the change-ontology bridge has no explanatory remainder.
