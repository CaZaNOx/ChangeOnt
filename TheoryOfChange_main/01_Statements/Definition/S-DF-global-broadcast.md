# Global broadcast — distributing the specious present across workspaces
## Claim (formal)
Define a broadcast operator \(\mathcal{B}:\mathcal{X}\to\mathcal{W}\) that maps the time-kernel–filtered state \(\bar{x}_t\) into a global workspace \(W_t = \mathcal{B}(\bar{x}_t)\) shared across subsystems; track broadcast bandwidth \(B_t \in [0,1]\) as reachability of that workspace.

## Philosophical Translation (of formal claim)
“Global availability” is made concrete: a shared buffer of the specious present with measurable bandwidth, not a metaphor.

## Philosophical Justification
- [[S-DF-time-kernel]] defines the specious present; broadcasting it provides coherence across subsystems/agents.
- Making broadcast explicit and measurable avoids hand-wavy “workspace” claims and supports audits of attention/focus distribution.

## Clarifications / Further Context
- Invariance of the broadcast (what stays consistent across recipients) and logged reach \(B_t\) allow comparing attention/focus across agents.
- Applicable to multi-agent or multi-module settings; bandwidth limits should be tracked as part of SE budgeting.
