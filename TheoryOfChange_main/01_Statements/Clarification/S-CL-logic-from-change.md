# Logic arises as invariance under change; classical logic is the zero-change limit

AI_13 sketches the derivation: truth = invariance under admissible change. Build logic by defining the invariance-refinement operator, modal/dynamic operators (eventually, always) as re-labeled versions of “again/from now on,” and show how path categories/topoi glue local invariances into global ones. Classical logic appears as the limit where change freezes (ρ→0), so excluded middle (and other classical laws) are special regimes, not primitives. This ties logic to change directly rather than importing a pre-built formal calculus.
