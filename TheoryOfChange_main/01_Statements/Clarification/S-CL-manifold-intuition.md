# Manifolds look flat locally but can curve globally

AI_13 explains manifolds in plain language: they are single spaces that, around every point, admit a patch (chart) diffeomorphic to \(\mathbb{R}^n\); patches overlap with smooth transition maps. This lets us treat complicated global shapes (globe, torus, configuration spaces) as “locally simple, globally whatever.” The metric/geometry is an extra layer: the same manifold can host different Riemannian metrics, so curvature comes from the chosen metric, not from the manifold structure itself.
