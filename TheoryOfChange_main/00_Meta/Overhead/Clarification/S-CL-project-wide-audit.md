# Whole-project audit lists change calculus tasks and Gödel operator requirements

The broader project review (lines 19045‑19087) extends the critique across all Change Ontology materials: solid ES/SoE/CoE scaffolding and geometry/dynamics bridges exist, but reviewers will demand a formal change calculus (operators, composition, invariants), a type system for structural contradictions, explicit bridge laws from latent manifolds to subjective reports, and an operational Gödel-hole operator (persistent residuals that vanish only when representation expands). Action items include anchoring charts to tasks, deriving metrics from Fisher info, preregistering predictions (geodesic length ↔ RT, curvature peaks ↔ interference, topology changes ↔ insight), and delivering toy proofs plus experiments that show residual collapse only when new latents are added.
