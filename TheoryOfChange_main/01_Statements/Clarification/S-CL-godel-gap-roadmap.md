# Gödel-gap roadmap links hedge/probe/bypass, probes, and variable spawn to evaluation

AI_13’s later sections turn the Gödel toolkit into a field roadmap: define gap scores \(\Gamma_t = \Delta H_t + \lambda \cdot \text{BF}_t\), run disagreement-driven probes in high-gap regions, log gap density \(\gamma\) to allocate resources, and only spawn variables when structured gaps persist (MDL/Bayes-factor gains). The program compares adaptive models (gap-triggered latent births, probe policies) against baselines, shows faster shock recovery (~46% error drop), and sets falsifiers (if tuned fixed models match, the Gödel-gap mechanism fails). It also extends the story to science/markets/contextuality: paradigms as attractors, anomalies as Gödel gaps, contextuality gaps in quantum logic, etc. This note captures the deliverable-level plan: measure, probe, spawn, and audit gaps with concrete metrics and benchmarks.
