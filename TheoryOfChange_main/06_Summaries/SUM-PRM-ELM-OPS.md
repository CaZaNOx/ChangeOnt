# Primitives, Elements, and Operators — overview

Scope
- Brief map from PRM (1–15) to key ELM and OPS.

Core nodes
- PRM: bend (P1), gauge (P2), MDL (P3), ReID (P4), temporal ops (P5), change ops (P6), precision (P7), loopiness (P8), variable birth (P9), changeops core (P10), residuation (P11), closure (P12), creative option birth (P13), depth/breadth flip (P14), order/arisal (P15)
- ELM: EA (HAQ), EB (GHVC), EC (Identity), ED (Gauge‑Warp), EE (Compressibility), EF (Router), EG (Density/Precision), EH (Breadth/Depth), EI (Change Operators), EJ (Order Asymmetry)
- OPS: J1 (Bend Substitution), J2 (Quotient Classes), J3 (Attention Warp), J4a (ReID Closure), J4b (Counterfactual Bend)

Key takeaways
- PRMs provide the measurable/control knobs; ELMs orchestrate behaviors; OPS implement reusable transforms.
