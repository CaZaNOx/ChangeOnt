# Gödel gaps trigger new attractors and novelty passes

AI_13 links Gödel-like gaps to attractor dynamics: when exploration repeatedly hits a wall (a Gödel gap/constraint), the system can only lower cost by inventing a new move, birthing a fresh attractor. Identity is a self-maintaining groove; novelty/arisal is a high-gain pass that opens plasticity; Gödel-like holes mark where the current geometry cannot absorb perturbations, so new coordinates appear. This ties Gödel pressure directly to attractor creation and change fitness.
