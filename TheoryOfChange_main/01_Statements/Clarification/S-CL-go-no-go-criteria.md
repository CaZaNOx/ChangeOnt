# Establish go/no-go thresholds for the gauge-alignment program

Go if cross-subject transport cuts predictive error ≥50% relative to no-alignment and ≥25% relative to any non-\(J\) baseline, if \(J\)-component lesions induce the preregistered degradation patterns, and if gauge residue remains small and stable across tasks. No-go if alignment works only inside one task family, invariants fail to generalize, non-conscious agents pass the entire battery, or gauge residue balloons. Having explicit thresholds prevents endless tweaking and forces honest evaluation of whether the phenomenology program is succeeding.
