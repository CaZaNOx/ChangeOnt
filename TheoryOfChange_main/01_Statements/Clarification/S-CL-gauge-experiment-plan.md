# Build canonical gauge group experiments for experience manifolds

The “do-this-next” package (lines 17220‑17272) specifies the formal object needed for a gauge story: for each subject \(i\) learn an experience manifold \(M_i\), canonical latent \(Z^*\), and warps \(W_i\) aligning subjects up to gauge group \(G\) (smooth reparameterizations that leave observables invariant). Measure invariants—\(d\) (distance metrics), \(K_u\) (urgency curvature), \(A\) (alignment field), \(H\) (loop holonomy), \(C\) (thick-present scalar)—via psychophysics, physiology, and loop integrals. Show that when two systems match on \(\{d,K_u,A,H,C\}\), no further fact remains. The experimental program: learn \(Z^*\) with contrastive/multiview ML, estimate \(W_i\) via diffeomorphic flows, probe invariants with lesion-like manipulations (dual-task, sleep deprivation, TMS) and cross-subject transport tests, and validate that counterfactuals commute once warped into the canonical gauge.
