# Consciousness requires broadcast gates, memory channels, and partition rules

AI_13 extends the operational story: Awareness occurs only when updates are high-precision and integrated into the self-loop (broadcast \(B_t\) × ownership \(\pi\)). Partition rules state that two self-loops count as distinct subjects when reciprocal information drops below a threshold while each remains self-consistent. Memory gate \(M\) is orthogonal to consciousness: you can be conscious of a stimulus yet fail to encode it (e.g., reading-while-driving); the model predicts how SSS (social/task relevance), WWW (attention bandwidth), and urgency reshape the gate. Pathologies (mind-wandering, slips, analgesia) arise from specific gate settings. Tool ownership and broadcast signatures (e.g., P3b) are empirical hooks. This note consolidates those gating/partition conditions so experiments can target the measurable levers.
