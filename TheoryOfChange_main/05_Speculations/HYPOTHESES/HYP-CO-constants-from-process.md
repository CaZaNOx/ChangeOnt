# Hypothesis — CO constants as process agreements (π/e lens)

Claim (speculative)
- Certain constants (π, e) can emerge as “agreement scalars” from purely operational data on change (bend costs, spread, flux) without assuming Euclidean structure, if the medium satisfies specific regularity (e.g., Dirichlet-form setting, spectral dimension ~2, doubling + Poincaré). The same pipeline in anisotropic or nonlocal media should drift away from the classical constants.

Test sketch
- Work in metric-measure spaces with a conservative symmetric Markov semigroup (heat kernel). Define three operational meters (spread decay, flux closure, spectral density) from data without Fourier/Lebesgue priors; estimate the scalar that equalizes them. Sweep media: Euclidean grid, Lévy flights (α<2), anisotropic diffusion, sphere/hyperbolic, random geometric graphs. Record convergence or drift of the scalar.

Falsifiers / risks
- Scalar depends on preprocessing (windowing, frames) → not invariant.
- Only appears when Euclidean structure is smuggled in → trivializes the claim.
- No uniqueness or stability under finite-sample bounds → useless as an invariant.

Why it matters (if true)
- Provides a CO-native diagnostic for spectral dimension/isotropy instead of hand-waving “π appears”. If false, it reins in overextended “constants from change” narratives.
