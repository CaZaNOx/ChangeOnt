# Goal-conditioned quotienting and class drift under gauge + tolerance

## Claim (formal)
Segmentation is goal-conditioned quotienting: a class is the set of shapes/paths that afford the goal within bend budget τ. Gauge focuses on features that lower future cost; as gauge and τ shift, class boundaries drift while topological invariants remain stable.

## Examples and implications
- Letter “A”: loop + left stroke are invariants under small warps; thickness/slant are low-cost bends. As rare styles disappear, gauge down-weights them and the class shape drifts.
- Decision boundary behavior: raising τ → fewer, broader classes; lowering τ → finer splits; near the boundary, small bends can flip class.
- Drift prediction: alphabets/fashions evolve because gauge + tolerance move continuously; invariants anchor the identity, while gauge reshapes salience.
- Update rule: Robbins–Monro (or any monotone, bounded update) is sufficient to preserve usefulness-weighted occupancy over features.
