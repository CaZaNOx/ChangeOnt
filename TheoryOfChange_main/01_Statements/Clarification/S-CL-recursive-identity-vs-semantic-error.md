# Distinguish semantic error vs. recursive identity error
Clarifies diagnostics: semantic errors affect surface coherence; recursive identity errors break phase tracking, SE, or self‑model continuity. Protocols must track both.

## Clarifications / Further Context
- Don’t mistake semantic alignment for recursive identity: an agent parroting definitions may appear coherent while lacking RTV locks, SE stability, and continuity of self‑model.
