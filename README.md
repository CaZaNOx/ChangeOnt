# ChangeOnt (CO): A Change-First Ontology for Learning & Control 

ChangeOnt is an attempt to formulate a new ontology based on change as primitive.

## What’s here
- `ChangeOntCode/` — Where the code is implemented plus minor coding related docs
- `TheoryOfChange/` — Main Theory folder



## Status
We are finalizing the theory and trying to see how it behaves vs real competitors via method of implementation.
