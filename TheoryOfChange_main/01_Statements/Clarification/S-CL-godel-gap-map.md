# Simulate Gödel-pressure, option-preserving policies, and gap maps

AI_13 sketches experiments: run small simulations where Gödel-pressure triggers variable births under resource budgets; log option-preserving choices near model gaps (hedge/probe/invent); draw Gödel-gap maps showing attractor basins, bend tolerances, and “G?” regions; and use these maps to de-risk real decisions (hedge, probe, bypass) depending on hole density. This makes Gödel-gap talk concrete for planners and provides templates for visual/interactive artifacts.
