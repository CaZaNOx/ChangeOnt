# Flow state is a high-presence, focused Φ configuration

Flow feels like laser focus, smooth action, altered time, and low rumination. In CO terms it is a transient attractor: prediction errors stay small and informative, presence \(P\) is high, salience is tightly focused, valence/urgency sit in the sweet zone (motivating but not anxious), and working memory serves the task instead of self-talk. Context switches or interruptions collapse the basin. This links the subjective description to the Φ invariants.
