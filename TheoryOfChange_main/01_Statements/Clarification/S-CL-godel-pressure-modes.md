# Gödel pressure is model-evidence deficit that forces class expansion

AI_13 distills Gödel pressure into three equivalent operational stories: (1) sustained model-evidence deficits that cannot be closed inside the current model class trigger _model class change_ (new latents/policies); (2) subcategory non-closure forces a refinement functor—it’s a categorical signal that new objects/morphisms are required; (3) no-control regimes (no policy in the present class lowers free energy) tilt the trade-off toward expanding the class despite cost. To exploit the idea, pin precise primitives (Change, Stability, Attractor, Urgency, Gödel-pressure), specify target domains (cognitive science, complex systems), and define where Gödel-hole density offers leverage. This frames Gödel pressure as a quantifiable innovation indicator, not a poetic placeholder.
