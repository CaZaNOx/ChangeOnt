# Map prediction-error types to phenomenology across channels

To finish operational closure AI_13 proposes mapping prediction-error _types_ (not just magnitudes) into experiential axes: frustration vs boredom vs relief tracks different appraisals, each with distinct coupling to physiology (pupil, HRV, EDA), behavior, and report structure. Fit functions \(g,h,\Lambda\) on person A, transport to person B via gauge-limited warps, and verify that cross-subject mappings respect these typed errors. The manifold is built from multi-view data (reports, behavior, physiology) so that the same structural error yields the same felt geometry after alignment.
