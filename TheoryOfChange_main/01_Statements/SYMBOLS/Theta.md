# θ — Threshold

Definition (formal):
Critical value used in identity and validation predicates (e.g., Sim≥θ) to determine acceptance under declared tolerances (ε, σ(ε)).

## Example
- Identity recognition if \(\frac{1}{|K|}\sum_{k\in K} Sim(T_k(P),P) \geq \theta\).

## Clarifications
- θ is context‑declared; not universal.
