# Rigidity lemmas block zombies, inverted qualia, and Mary

Once the perspective gauge is fixed, phenomenology can’t be altered without changing control invariants. Zombies would need \(I\) and \(\Phi\) identical yet “no experience”—mere relabeling, because behavior and counterfactual control change when \(P\) or \(I\) collapse (as under anesthesia). Inverted qualia would require an automorphism that flips the action-stabilized qualia manifold; no such symmetry survives the gauge constraints (constancy tasks break). Mary lacked the gauge—her loop never instantiated the color manifold; when it does, \(\Phi\) enters a new regime. Thus the explanatory gap reduces to specifying \(I+\Phi\).
